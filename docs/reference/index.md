# Package index

## Models and biplots for multi-environment trials

Functions to fit the AMMI and SREG models and their biplots.

- [`rAMMIModel()`](https://github.com/jangelini/geneticae/reference/rAMMIModel.md)
  : Robust AMMI Model

- [`rAMMIPlot()`](https://github.com/jangelini/geneticae/reference/rAMMIPlot.md)
  : AMMI Biplots with ggplot2

- [`rSREGModel()`](https://github.com/jangelini/geneticae/reference/rSREGModel.md)
  : Site Regression model

- [`rSREGPlot()`](https://github.com/jangelini/geneticae/reference/rSREGPlot.md)
  :

  GGE biplots with ggplot2

## Missing data imputation

Methods for the imputation of missing data in multi-environment trials.

- [`imputation()`](https://github.com/jangelini/geneticae/reference/imputation.md)
  : Imputation of missing cells in two-way data sets

## Datasets

Sample datasets (see the

  Clones from the PLRV population
